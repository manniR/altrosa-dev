## Installation

Install NPM-Modules via 

```
$ npm install
# OR
$ yarn
```

Then run

```
# Development initialization
$ npm run init

# Develop-Mode with file-watch
$ npm run dev

# Copy HTML-Assets and Images
$ npm run uiimages && npm run htmlassets

# Build for production
$ npm run build
```
